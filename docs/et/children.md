# Utah-Vid-ia lastele (ja uudishimulikele täiskasvanutele)

Kujuta ette, et sinu arvutis on **ülimalt kiire matemaatikaabiline** nimega **GPU**. See lahendab miljoneid matemaatikaülesandeid korraga — suurepärane videomängude ja tehisintellekti jaoks.

## Probleem: üks eratee

Praegu saavad paljud programmid hõlpsalt kasutada ainult ühe ettevõtte GPU-d (tihti NVIDIA). See on nagu linn, kus iga auto **peab** sõitma ühe ettevõtte erateel. Kui ostad teise auto (teise GPU-margi), ütleb tee *„vabandust, keelatud.”*

See pole õiglane ja see maksab peredele ja koolidele rohkem raha.

## Mida Utah-Vid-ia teeb: maantee kõigile

**Utah-Vid-ia** ehitab **ühise maantee** erateede peale.

Sinu programm ütleb endiselt „liida need numbrid” või „korruta see maatriks.” Utah-Vid-ia:

1. **Kuulab** matemaatika taotlust (Ghost Kernel)
2. **Tõlgib** selle, et rohkem GPU-sid mõistaksid (Vector Compiler)
3. **Valib** klastris kõige vähem hõivatud GPU (Osmotic Router)
4. **Valmistab** järgmise töö ette, et midagi ei peaks ootama (ZEO Pre-Sight)
5. **Parandab** väikesed mäluvigastused õrnalt, selle asemel et kokku kukkuda (Entropy Shield)

## Lõbusad analoogiad

| Utah-Vid-ia osa | Päriselu idee |
|-----------------|---------------|
| Ghost Kernel | Tõlk uksel |
| Vector Compiler | Rätsep, kes kohandab riideid samal ajal kui sa kõnnid |
| Osmotic Router | Treener, kes saadab mängijad kõige vähem väsinud meeskonnakaaslase juurde |
| ZEO Pre-Sight | Järgmise lehe lugemine, samal ajal kui lõpetad praeguse |
| Photonic Bridge | Taskulambid, mis vilgutavad sõnumeid, selle asemel et lärmakas toas karjuda |

## Proovi (täiskasvanuga)

```bash
pip install -e ".[dev]"
utahvidia
```

Näed kuut demo käivitumist. Igaüks prindib, mida Utah-Vid-ia parasjagu teeb.
