# Поддержать Utah-Vid-ia

Utah-Vid-ia — проект с открытым исходным кодом (MIT). Пожертвования помогают финансировать документацию, бенчмарки, тестирование оборудования и нативную работу с ядрами на GPU NVIDIA, AMD и Intel.

## На что идут пожертвования

- Бенчмарк-стенды с GPU разных вендоров (потребительские и датацентровые)
- CI-раннеры с CUDA toolchain
- Поддержка переведённой документации
- Прототипное оборудование фотонного моста (узлы ESP32)
