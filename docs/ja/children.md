# 子ども（と好奇心旺盛な大人）向け Utah-Vid-ia

コンピュータの中には **GPU** という、とても速い計算のお手伝いさんがいます。何百万もの計算を一度にこなせるので、ゲームや AI にぴったりです。

## 問題：ひとつの私有道路

今は、多くのプログラムが特定の会社の GPU しか簡単に使えません（よく NVIDIA）。町に例えると、すべての車が **必ず** ある会社の私有道路を走らなければならないようなものです。別の車（別ブランドの GPU）を買っても、道路は *「ごめん、ダメ」* と言います。

それは不公平ですし、家庭や学校にもっとお金がかかります。

## Utah-Vid-ia がすること：みんなのための高速道路

**Utah-Vid-ia** は、私有道路の上に **みんなが使える高速道路** を作ります。

プログラムはこれまでどおり「この数字を足して」「この行列を掛けて」と言います。Utah-Vid-ia は：

1. 計算のリクエストを **聞き取る**（Ghost Kernel）
2. より多くの GPU が理解できるように **翻訳する**（Vector Compiler）
3. クラスターで一番暇な GPU を **選ぶ**（Osmotic Router）
4. 次の仕事を早めに **準備する** ので待ち時間が減る（ZEO Pre-Sight）
5. 小さなメモリの不具合をクラッシュさせずに **やさしく直す**（Entropy Shield）

## たのしいたとえ

| Utah-Vid-ia の部品 | 身近なイメージ |
|------------------|----------------|
| Ghost Kernel | 入り口の通訳さん |
| Vector Compiler | 歩きながら服のサイズを直してくれる仕立て屋 |
| Osmotic Router | いちばん疲れていない選手にパスを出すコーチ |
| ZEO Pre-Sight | 今のページを読み終える前に次のページを読んでおくこと |
| Photonic Bridge | うるさい部屋で叫ぶ代わりに、懐中電灯で合図を送ること |

## 試してみる（大人と一緒に）

```bash
pip install -e ".[dev]"
utahvidia
```

6つのデモが動きます。それぞれ Utah-Vid-ia が何をしているか表示されます。
