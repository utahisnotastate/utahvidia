| ティア | 推奨 | 特典 |
|------|-----------|----------|
| サポーター | $5+ | Pro プロファイル + パトロン CLI |
| エンスージアスト | $15+ | + 任意 [PATRONS.md](../../PATRONS.md) クレジット |
| レジェンド | $50+ | + 早期 `@patron` リリースタグ |

**完全ガイド:** [GPU アンロック パトロン・プログラム](gpu-unlock-patron.md) · [ゲーマーチュートリアル](gaming-tutorial.md)
