# Utah-Vid-ia を支援する

Utah-Vid-ia は**無料のオープンソース（MIT）**です。すべてのゲーミング機能がアンロック済み — 支払い不要。

## PayPal（任意）

**メール:** `utah@utahcreates.com`

**寄付:** [paypal.com/donate](https://www.paypal.com/donate/?business=utah%40utahcreates.com)

寄付は GPU ベンチマーク、翻訳、CUDA カーネル作業、Windows exe ビルドに充てられます。

## ゲーマー

無料のゲーミングアンロックをダウンロード — 寄付不要:
