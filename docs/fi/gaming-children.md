# GPU:n tulevaisuus lapsille — lopetammeko grafiikkakorttien ostamisen?

## Laserkaupunki

Kuvittele valtava kaupunki (sinun **GPU**) tehtaineen, jotka piirtävät pelikuvia. Tänään paketit kulkevat **rekoilla** (elektronit kuparijohdoissa). Rekot jumittuvat liikenteeseen ja kuumenevat.

Kaukana tulevaisuudessa käytämme **lasertubeja** — valo kuljettaa paketit heti lähes ilman lämpöä. Tehtaat voivat **muuttaa muotoaan** joka mikrosekunti sen mukaan, mitä niiden täytyy rakentaa.

## Sarjakuvataiteilija

Pelit piirtävät kuvia kuin sarjakuva, sivu kerrallaan. Vanhat järjestelmät pakottavat taiteilijan odottamaan hidasta lähettiä (**ajuri**) ennen sivun näyttämistä. Se aiheuttaa **nykimistä**.

**Utah-Vid-ia Latency Shield** asettaa taiteilijan viereen älykkään kopioimiskoneen. Se arvailee seuraavan sivun liikkeestä ja näyttää sujuvia sivuja odottamatta lähettiä.

## Lopetammeko GPU:iden ostamisen?

**Rehellinen vastaus:** Emme ikuisesti. Fysiikka asettaa katon — et voi ohjelmistopäivityksellä ylittää piisi lämpö- ja matemaattisia rajoja.

**Mutta voi tuntua siltä:** Tulevaisuuden ohjelmisto (kuten Utah-Vid-ia) lopettaa tehon tuhlaamisen asioihin, joita **silmäsi eivät näe**. Piirtää pienen sumean kuvan, sitten tekoäly täydentää loput — kuin uneksija viimeistelee luonnoksen.

Utah-Vid-ia auttaa **vanhoja** kortteja pysymään hyödyllisinä pidempään käyttämällä jokaisen syklin viisaasti.
